# Resonate Frontend Demo

Quick Vite + React demo that recreates the Brain Activation results screen with mock data.

Run locally:

```bash
cd frontend
npm install
npm run dev
```

Open http://localhost:5173

Notes:
- This is a lightweight demo using mock JSON under `src/mock/result.json`.
- The Brain visualization draws 200 parcel points in two hemispheres and colors them by modality.
